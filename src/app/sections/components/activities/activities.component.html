Desarrollo aplicaciones modernas con un enfoque en Frontend, complementando con habilidades en Backend y bases de datos para crear soluciones completas.

Frontend

Diseño e implemento interfaces modernas y responsivas utilizando tecnologías como React, Angular y Tailwind CSS, asegurando experiencias de usuario fluidas.

Backend

Desarrollo servicios robustos y seguros utilizando Node.js con Express y Asp.net Web con Entity FrameWork , creando APIs RESTful y conectándolas con interfaces Frontend.

Bases de Datos

Diseño y optimización de bases de datos relacionales utilizando MySQL y MSSQL y bases no relacionales MongoDB, asegurando integridad y eficiencia en el manejo de datos.
